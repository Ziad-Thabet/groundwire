# Groundwire

AI-powered document intelligence SaaS — teams upload documents and get a RAG-based AI assistant that answers questions grounded in that content.

**Status:** 🚧 In active development (Phase 0)

## Stack

- Frontend: React + TypeScript + Tailwind + shadcn/ui
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL + Prisma + pgvector
- Queue: BullMQ + Redis
- LLM: Anthropic Claude API
